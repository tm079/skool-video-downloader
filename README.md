# Skool Video Downloader (Browser Extension)
A browser extension that adds a convenient download button to Skool (skool.com) classroom pages, allowing you to save videos for offline viewing when permitted by the content owner.

- Download accessible course videos for learning anytime, anywhere
- Backup important educational materials for personal use
- Build your own offline study library
- Avoid losing access to learning content due to connection issues or account limitations
